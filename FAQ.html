Perguntas frequentes:

Quais os requisitos para ser um afiliado?
Para fazer parte da nossa rede de afiliados você precisa ter mais de 18 anos de idade, além de uma rede de
contatos ou canal de divulgação, conseguindo assim recrutar novos usuários para a VemBetar.

Como eu posso usar meu link de afiliado?
Ao se cadastrar no nosso programa de afiliados, você receberá um link exclusivo na sua conta, para divulgar
como preferir, nas suas redes sociais, no seu site, no seu canal, blog, etc.

Qual o valor mínimo para receber comissões?
O valor mínimo para realizar um pagamento é de R$ 500. Se o afiliado não atingir o pagamento, ele será
transferido para o mês seguinte, dessa forma até o afiliado conseguir completar o valor mínimo.

Como acompanho meus resultados?
Você poderá acompanhar todos seus resultados na nossa plataforma de gerenciamento, com seu usuário e senha criados.

Preciso manter um número mínimo de jogadores ativos na minha rede para continuar ganhando comissões?
Você deverá ter, pelo menos, 5 (cinco) jogadores ativos na plataforma, num período de três meses. Consideramos
jogadores ativos todos os jogadores que realizam apostas através de seu código promocional. A cada trimestre,
você deve acumular pelo menos mais cinco usuários ativos para se manter como nosso afiliado.

Como a comissão será calculada?
A comissão será calculada usando a seguinte fórmula:
Receita líquida de jogos (“NGR”) = receita bruta de jogos (“GGR”) – (impostos + pagamentos de bônus + estornos) – (taxa de serviços).

Como o pagamento será realizado?
O Pagamento é calculado todo mês, sobre a porcentagem dos ganhos e perdas (Revenue Share) dos jogadores.

Qual é o método de pagamento disponível?
A nossa plataforma utiliza o PIX como pagamento, já que é um método rápido e seguro. O pagamento é feito todo mês,
depois do mês corrente terminar, até 30 dias o afiliado vai receber a comissão.

Quantos usuários eu preciso indicar para receber o pagamento?
Para estar apto a receber as comissões todo mês o afiliado precisa indicar pelo menos 5 novos usuários.
O afiliado pode usar o seu link ou materiais de marketing disponíveis na plataforma.
